# Additional Resources

A curated collection of resources to deepen your understanding of AI agents and continue your learning journey.

## Papers and Research

### Foundational Papers

- **[ReAct: Synergizing Reasoning and Acting in Language Models](https://arxiv.org/abs/2210.03629)** (2022)
  - Introduces the ReAct pattern for interleaving reasoning and action
  
- **[Chain-of-Thought Prompting Elicits Reasoning in Large Language Models](https://arxiv.org/abs/2201.11903)** (2022)
  - Demonstrates how to improve reasoning through step-by-step prompting

- **[Tree of Thoughts: Deliberate Problem Solving with Large Language Models](https://arxiv.org/abs/2305.10601)** (2023)
  - Explores multiple reasoning paths for complex problem-solving

- **[Reflexion: Language Agents with Verbal Reinforcement Learning](https://arxiv.org/abs/2303.11366)** (2023)
  - Self-reflection and learning from mistakes

### Tool Use and Function Calling

- **[Toolformer: Language Models Can Teach Themselves to Use Tools](https://arxiv.org/abs/2302.04761)** (2023)
  - How LLMs can learn to use external tools

- **[Gorilla: Large Language Model Connected with Massive APIs](https://arxiv.org/abs/2305.15334)** (2023)
  - Training LLMs to use APIs effectively

### Multi-Agent Systems

- **[Communicative Agents for Software Development](https://arxiv.org/abs/2307.07924)** (2023)
  - ChatDev: Multi-agent collaboration for software engineering

- **[AutoGen: Enabling Next-Gen LLM Applications via Multi-Agent Conversation](https://arxiv.org/abs/2308.08155)** (2023)
  - Microsoft's framework for multi-agent systems

## Frameworks and Libraries

### Agent Frameworks

- **[LangChain](https://python.langchain.com/)**
  - Comprehensive framework for building LLM applications
  - Extensive tool ecosystem
  - [Documentation](https://python.langchain.com/docs/get_started/introduction)

- **[LlamaIndex](https://www.llamaindex.ai/)**
  - Data framework for LLM applications
  - Excellent for RAG implementations
  - [Documentation](https://docs.llamaindex.ai/)

- **[AutoGen](https://microsoft.github.io/autogen/)**
  - Microsoft's multi-agent conversation framework
  - Built-in code execution
  - [Documentation](https://microsoft.github.io/autogen/docs/Getting-Started)

- **[CrewAI](https://www.crewai.com/)**
  - Role-based multi-agent framework
  - Simplified orchestration
  - [Documentation](https://docs.crewai.com/)

- **[Semantic Kernel](https://learn.microsoft.com/en-us/semantic-kernel/)**
  - Microsoft's SDK for AI orchestration
  - Enterprise-focused
  - [Documentation](https://learn.microsoft.com/en-us/semantic-kernel/overview/)

### Vector Databases

- **[Chroma](https://www.trychroma.com/)** - Open-source embedding database
- **[Pinecone](https://www.pinecone.io/)** - Managed vector database
- **[Weaviate](https://weaviate.io/)** - Open-source vector search engine
- **[Qdrant](https://qdrant.tech/)** - Vector similarity search engine
